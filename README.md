# PROYECTO3I2CNEOPIX
Proyecto 3 de Electrónica Digital 2 utilizando comunicación I2C, almacenamiento SD y la pantalla ILI9341 TFT LCD con los microcontroladores ESP32 & Tiva C.
Por lo que para visualizar el código del ESP32 se debe dirigir a la carpeta I2C SENSOR - ESP32 → src → main.cpp. Mientras que para visualizar el código de la Tiva C dirigirse a la carpeta TEST 2023-11-13_0515 → TEST → TEST.ino
